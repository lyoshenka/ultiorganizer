Register name

Ultiorganizer user register

Purpose of the register

  • System user rights.
  • Players accreditation.
  • Scorekeeping and statistics.
  • Season announcements.

Register content

  • Name
  • E-mail
  • Gender
  • Birthtime
  • Birthplace
  • Nationality
  • Jersey number
  • Game statistics: games, assist, goals.
  • Hand
  • Height
  • Weight
  • Picture
  • Player description
  • Player achievements
  • Username
  • Password (encrypted)
  • User rights

Data sources

All data is provided by team contact person or user itself. Player statistics are collected from game score sheets inserted in system by score keepers.

Data export

Player details and statistics are freely available in public pages. User authentication data is not given for 3rd party.

Data transfer outside EU or ETA

Data register is not given outside Finland.

Data protection principles

Analog data: There is no analog version of register.

Digital data: Access to register data is limited with user authentication. Good security principles are followed with data protection.
